# Scuba Cat Vision Trigger

A fun, interactive Computer Vision project that detects specific, complex body gestures to instantly trigger the viral "Scuba Cat" meme along with the legendary "Kicau Mania" soundtrack! 

## About The Project

Ever wanted to summon a meme in real life using just your movements? This project acts as a high-precision spatial state engine. It monitors your webcam feed in real-time and waits for the perfect combination of actions. 

When the exact conditions are met, it seamlessly injects `src/scuba.gif` into your video feed and blasts `src/music/lagu_kicau_mania.mp3`.

**The Secret Activation Gesture:**
1. **Right Hand:** Waving continuously left and right.
2. **Left Hand:** Brought up close to pinch/cover the nose.
3. **Torso:** Swaying slightly side-to-side in sync.

*Boom! The Scuba Cat appears.* 

## Tech & Tools
* Computer Vision (Spatial tracking & landmark detection)
* I used a **Gemini-CLI** to help build the logic and prompt architecture for this project!
